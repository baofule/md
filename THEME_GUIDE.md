## 🌤️ 天气系统详解

### ☀️ 晴天 (Sunny)
```
背景: 明亮的蓝色渐变
天体: 太阳（带脉冲动画）
特效: 清新海浪
氛围: 活力、清新
```

### 🌧️ 雨天 (Rainy)
```
背景: 深灰色渐变
天体: 无（阴天）
特效: 雨滴下落动画
氛围: 沉静、思考
```

### ❄️ 雪天 (Snowy)
```
背景: 浅灰色渐变
天体: 月亮
特效: 雪花飘落动画
氛围: 寒冷、宁静
```

### 🌅 黄昏 (Sunset)
```
背景: 橙红色渐变
天体: 太阳（夕阳）
特效: 温暖海浪
氛围: 温暖、浪漫
```

### 🌙 夜晚 (Night)
```
背景: 深蓝色渐变
天体: 月亮（带光晕）
特效: 静谧海浪
氛围: 神秘、安静
```

## 🛠️ 自定义指南

### 修改天气颜色

在 `layouts/partials/header.html` 中修改天气背景样式：

```css
.weather-bg.sunny {
    background: linear-gradient(180deg, #0ea5e9 0%, #38bdf8 50%, #7dd3fc 100%);
}
```

### 调整海浪动画

修改 `<svg class="waves">` 中的动画属性：

```xml
<animate attributeName="d" dur="10s" repeatCount="indefinite"
```

- `dur="10s"` - 动画持续时间（调整这个改变速度）
- `repeatCount="indefinite"` - 无限循环

### 自定义雨滴/雪花

在 `static/js/theme.js` 中修改粒子数量：

```javascript
// 雨滴数量
const dropCount = 100;

// 雪花数量
const flakeCount = 50;
```

### 修改卡片样式

在 `layouts/partials/header.html` 中修改 `.glass-card` 样式：

```css
.glass-card {
    background: var(--glass-bg);
    backdrop-filter: blur(20px);  /* 模糊程度 */
    border: 1px solid var(--glass-border);
    border-radius: 20px;           /* 圆角大小 */
    padding: 2rem;                 /* 内边距 */
}
```
