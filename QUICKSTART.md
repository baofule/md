# 🚀 快速开始

## 🎉 恭喜！你的小说网站已经就绪！

**网站地址**: http://localhost:3000/md/

## ✨ 当前配置

- **主题**: Paper (GitHub 3000+ Stars 简约主题)
- **颜色**: 蓝色主题
- **章节数**: 24章
- **服务器**: 运行在 3000 端口

## 🌟 Paper主题特色

### 极简设计
- ✅ 干净的排版
- ✅ 清爽的配色
- ✅ 简约的风格
- ✅ 完美的响应式
- ✅ 快速的加载

### 颜色主题
可选的6种颜色：

- ✅ 文章搜索

## 🎨 切换颜色主题

编辑 `config.toml`：

```toml
[params]
  color = "blue"  # 改成你喜欢的颜色
```

可选项：`light`, `dark`, `red`, `green`, `orange`

## 📖 查看网站

打开浏览器访问: **http://localhost:3000/md/**

## 🔄 更新章节

```bash
# 1. 在 没钱修什么仙/ 目录添加新章节
# 2. 运行更新脚本
./update.sh
```

## 🛠️ 常用命令

```bash
# 本地预览
hugo server

# 构建网站
hugo --minify

# 部署到GitHub
./deploy.sh
```

## 📂 文件说明

- `config.toml` - 网站配置
- `convert_novel.py` - 章节转换脚本
- `update.sh` - 快速更新脚本
- `deploy.sh` - 部署脚本
- `content/` - 网站内容
- `themes/paper/` - Paper主题
